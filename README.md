# 单词听写网站

一个基于 React + TypeScript + Vite 的单词听写学习工具。

## 功能特点

- 🎯 单词听写练习
- 📝 学习进度追踪
- 🎨 现代化 UI 设计
- 📱 响应式布局

## 技术栈

- **框架**: React 18
- **语言**: TypeScript
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **UI 组件**: shadcn/ui

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

构建后的文件位于 `dist` 目录，可直接部署到任何静态托管服务（GitHub Pages、Vercel、Netlify 等）。
